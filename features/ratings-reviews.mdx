## Overview

BeanQuick implements a comprehensive rating and review system that allows customers to rate products on a 1-5 star scale with optional text comments. Reviews are tied to delivered orders to ensure authenticity.

## Rating Model Structure

### Database Schema

```sql
CREATE TABLE calificaciones (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT NOT NULL,
    pedido_id BIGINT NOT NULL,
    producto_id BIGINT NOT NULL,
    estrellas INT NOT NULL COMMENT 'Del 1 al 5',
    comentario TEXT NULL,
    created_at TIMESTAMP,
    updated_at TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (pedido_id) REFERENCES pedidos(id) ON DELETE CASCADE,
    FOREIGN KEY (producto_id) REFERENCES productos(id) ON DELETE CASCADE
);
```

## Creating Reviews

### Submit Rating

Customers can rate products from delivered orders.

**Endpoint:** `POST /api/cliente/calificar`

**Request:**
```json
{
  "pedido_id": 15,
  "producto_id": 5,
  "estrellas": 5,
  "comentario": "Excellent cappuccino! Will order again."
}
```

**Implementation:**
```php
// CalificacionController.php:18
public function store(Request $request): JsonResponse
{
    // Validation
    $request->validate([
        'pedido_id'   => 'required|exists:pedidos,id',
        'producto_id' => 'required|exists:productos,id',
        'estrellas'   => 'required|integer|min:1|max:5',
        'comentario'  => 'nullable|string|max:255',
    ]);

    try {
        // 1. Verify order ownership and status
        $pedido = Pedido::where('id', $request->pedido_id)
                        ->where('user_id', Auth::id())
                        ->first();

        if (!$pedido) {
            return response()->json(['message' => 'Pedido no encontrado.'], 404);
        }

        // RULE: Can only rate delivered orders
        if (strtolower($pedido->estado) !== 'entregado') {
            return response()->json([
                'message' => 'Solo puedes calificar productos de pedidos entregados.'
            ], 403);
        }

        // 2. Prevent duplicate ratings
        $existe = Calificacion::where('pedido_id', $request->pedido_id)
                              ->where('producto_id', $request->producto_id)
                              ->exists();

        if ($existe) {
            return response()->json([
                'message' => 'Ya has calificado este producto.'
            ], 400);
        }

        // 3. Create rating
        $calificacion = Calificacion::create([
            'user_id'     => Auth::id(),
            'pedido_id'   => $request->pedido_id,
            'producto_id' => $request->producto_id,
            'estrellas'   => $request->estrellas,
            'comentario'  => $request->comentario,
        ]);

        return response()->json([
            'message' => '¡Gracias por tu calificación! ⭐',
            'calificacion' => $calificacion
        ], 201);

    } catch (\Exception $e) {
        return response()->json([
            'message' => 'Error al guardar la calificación',
            'error' => $e->getMessage()
        ], 500);
    }
}
```

**Success Response:**
```json

    "user_id": 3,
```

## Rating Business Rules

### 1. Delivered Orders Only

Customers can only rate products from orders with `estado = 'Entregado'`:

```php
if (strtolower($pedido->estado) !== 'entregado') {
    return response()->json([
        'message' => 'Solo puedes calificar productos de pedidos entregados.'
    ], 403);
}
```

This ensures all reviews are from verified purchases.

### 2. One Rating Per Product Per Order

Prevents duplicate ratings:

```php
$existe = Calificacion::where('pedido_id', $request->pedido_id)
                      ->where('producto_id', $request->producto_id)
                      ->exists();

if ($existe) {
    return response()->json(['message' => 'Ya has calificado este producto.'], 400);
}
```

### 3. Order Ownership

Users can only rate products from their own orders:

```php
$pedido = Pedido::where('id', $request->pedido_id)
                ->where('user_id', Auth::id())
                ->first();
```

### 4. Star Range

Ratings must be between 1 and 5 stars:

```php
'estrellas' => 'required|integer|min:1|max:5',
```

### 5. Optional Comment

Customers can leave text feedback (max 255 characters) or rate without comment:

```php
'comentario' => 'nullable|string|max:255',
```

## Viewing Reviews

### Get Product Reviews

Public endpoint to view all reviews for a product.

**Endpoint:** `GET /api/productos/{id}/calificaciones`

```php
// CalificacionController.php:83
public function porProducto($productoId): JsonResponse
{
    $calificaciones = Calificacion::where('producto_id', $productoId)
        ->with('usuario:id,name')  // Include reviewer name only
        ->orderBy('created_at', 'desc')
        ->get();

    return response()->json($calificaciones);
}
```

## Average Rating Calculation

Product average ratings are calculated dynamically:

```php
// ProductoController.php:167 (destacados endpoint)
$promedio = $producto->calificaciones->avg('estrellas');
$producto->calificaciones_avg_estrellas = $promedio ? round($promedio, 1) : 0;
```

**Example Calculation:**
- Review 1: 5 stars
- Review 2: 4 stars
- Review 3: 5 stars
- **Average:** (5 + 4 + 5) / 3 = 4.7 stars

## Featured Products

The home page displays top-rated products based on average ratings:

**Endpoint:** `GET /api/productos/destacados`

```php
// ProductoController.php:157
$destacados = $productos->map(function ($producto) {
    $promedio = $producto->calificaciones->avg('estrellas');
    $producto->calificaciones_avg_estrellas = $promedio ? round($promedio, 1) : 0;
    return $producto;
})
->filter(fn($p) => $p->calificaciones_avg_estrellas > 0)  // Only rated products
->sortByDesc('calificaciones_avg_estrellas')             // Highest first
->take(4)                                                 // Top 4
->values();
```

Only products with at least one rating are featured.

## Privacy Considerations

### Limited User Information

When displaying reviews publicly, only the reviewer's name is exposed:

```php
->with('usuario:id,name')  // Only id and name, not email or other data
```

### Owner-Only Modification

Customers can only update/delete their own reviews. The system should verify:

```php
$calificacion = Calificacion::where('id', $id)
    ->where('user_id', Auth::id())
    ->first();

if (!$calificacion) {
    return response()->json(['message' => 'No encontrado'], 404);
}
```
