## Get Featured Products

<Info>
This endpoint is public and does not require authentication.
</Info>

```http
GET /api/productos/destacados
```

Retrieve the top 4 highest-rated products across all businesses. Returns only products that have received at least one rating.

### Response

<ResponseField name="productos" type="array">
  Array of top-rated product objects
  
  <Expandable title="Product Object">
    <ResponseField name="id" type="integer">
      Product ID
    </ResponseField>
    
    <ResponseField name="nombre" type="string">
      Product name
    </ResponseField>
    
    <ResponseField name="precio" type="float">
      Product price
    </ResponseField>
    
    <ResponseField name="imagen" type="string">
      Image file path
    </ResponseField>
    
    <ResponseField name="empresa_id" type="integer">
      Business ID that sells this product
    </ResponseField>
    
    <ResponseField name="calificaciones_avg_estrellas" type="float">
      Average rating rounded to 1 decimal (1-5 stars)
    </ResponseField>
    
    <ResponseField name="empresa" type="object">
      Business information
      
      <Expandable title="Business Fields">
        <ResponseField name="id" type="integer">
          Business ID
        </ResponseField>
        
        <ResponseField name="nombre" type="string">
          Business name
        </ResponseField>
        
        <ResponseField name="logo" type="string">
          Logo file path
        </ResponseField>
        
        <ResponseField name="logo_url" type="string">
          Full URL to business logo
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET https://api.beanquick.com/api/productos/destacados \
  -H "Accept: application/json"
```
</RequestExample>

## Get Product Ratings

<Info>
This endpoint is public and does not require authentication.
</Info>

```http
GET /api/productos/{id}/calificaciones
```

Retrieve all customer ratings and reviews for a specific product.

### Path Parameters

<ParamField path="id" type="integer" required>
  Product ID
</ParamField>

### Response

<ResponseField name="calificaciones" type="array">
  Array of rating objects, sorted by most recent first
  
  <Expandable title="Rating Object">
    <ResponseField name="id" type="integer">
      Rating ID
    </ResponseField>
    
    <ResponseField name="user_id" type="integer">
      ID of the user who created the rating
    </ResponseField>
    
    <ResponseField name="pedido_id" type="integer">
      Order ID associated with this rating
    </ResponseField>
    
    <ResponseField name="producto_id" type="integer">
      Product ID
    </ResponseField>
    
    <ResponseField name="estrellas" type="integer">
      Star rating (1-5)
    </ResponseField>
    
    <ResponseField name="comentario" type="string">
      Optional review comment
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      When the rating was created
    </ResponseField>
    
    <ResponseField name="updated_at" type="string">
      When the rating was last updated
    </ResponseField>
    
    <ResponseField name="usuario" type="object">
      User information (limited fields for privacy)
      
      <Expandable title="User Fields">
        <ResponseField name="id" type="integer">
          User ID
        </ResponseField>
        
        <ResponseField name="name" type="string">
          User display name
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET https://api.beanquick.com/api/productos/15/calificaciones \
  -H "Accept: application/json"
```
</RequestExample>
