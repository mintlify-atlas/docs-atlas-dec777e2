<Warning>

</Warning>

## Create Rating

```http
POST /api/cliente/calificar
```

Create a rating for a product. Users can only rate products from their own orders, and only once per product. Duplicate ratings are prevented.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login
</ParamField>

### Body Parameters

<ParamField body="producto_id" type="integer" required>
  Product ID to rate
</ParamField>

<ParamField body="estrellas" type="integer" required>
  Star rating (1-5)
</ParamField>

<ParamField body="pedido_id" type="integer" optional>
  Order ID (optional)
</ParamField>

<ParamField body="comentario" type="string" optional>
  Review comment (optional)
</ParamField>

### Response

<ResponseField name="message" type="string">
  Success or error message
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.beanquick.com/api/cliente/calificar \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{
    "producto_id": 15,
    "estrellas": 5,
    "comentario": "Excelente calidad, muy recomendado"
  }'
```
</RequestExample>

<ResponseExample>
```json 200 Success
{
  "message": "Calificación guardada correctamente"
}
```

```json 403 Already Rated
{
  "message": "Ya calificaste este producto."
}
```

```json 422 Validation Error
{
  "message": "The given data was invalid.",
  "errors": {
    "estrellas": [
      "The estrellas must be between 1 and 5."
    ]
  }
}
```
</ResponseExample>

## List Customer Ratings

```http
GET /api/cliente/mis-calificaciones
```

Retrieve all ratings created by the authenticated customer, sorted by most recent first.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login
</ParamField>

### Response

<ResponseField name="calificaciones" type="array">
  Array of rating objects
  
  <Expandable title="Rating Object">
    <ResponseField name="id" type="integer">
      Rating ID
    </ResponseField>
    
    <ResponseField name="user_id" type="integer">
      Customer user ID
    </ResponseField>
    
    <ResponseField name="pedido_id" type="integer">
      Associated order ID
    </ResponseField>
    
    <ResponseField name="producto_id" type="integer">
      Product ID
    </ResponseField>
    
    <ResponseField name="estrellas" type="integer">
      Star rating (1-5)
    </ResponseField>
    
    <ResponseField name="comentario" type="string">
      Review comment
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      Creation timestamp
    </ResponseField>
    
    <ResponseField name="updated_at" type="string">
      Last update timestamp
    </ResponseField>
    
    <ResponseField name="producto" type="object">
      Product information including empresa relationship
      
      <Expandable title="Product Fields">
        <ResponseField name="id" type="integer">
          Product ID
        </ResponseField>
        
        <ResponseField name="nombre" type="string">
          Product name
        </ResponseField>
        
        <ResponseField name="imagen_url" type="string">
          Product image URL
        </ResponseField>
        
        <ResponseField name="empresa" type="object">
          Business that sells this product
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET https://api.beanquick.com/api/cliente/mis-calificaciones \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Accept: application/json"
```
</RequestExample>

<ResponseExample>

## Update Rating

```http
PATCH /api/cliente/calificaciones/{id}
```

Update an existing rating. Users can only update their own ratings.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login
</ParamField>

### Path Parameters

<ParamField path="id" type="integer" required>
  Rating ID to update
</ParamField>

### Body Parameters

<ParamField body="estrellas" type="integer" required>
  New star rating (1-5)
</ParamField>

<ParamField body="comentario" type="string" optional>
  New review comment (nullable)
</ParamField>

### Response

<ResponseField name="message" type="string">
  Success or error message
</ResponseField>

<RequestExample>
```bash cURL
curl -X PATCH https://api.beanquick.com/api/cliente/calificaciones/42 \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{
    "estrellas": 4,
    "comentario": "Actualicé mi opinión después de probarlo más veces"
  }'
```
</RequestExample>

<ResponseExample>
```json 200 Success
{
  "message": "Reseña actualizada correctamente"
}
```

```json 404 Not Found
{
  "message": "Reseña no encontrada"
}
```

```json 422 Validation Error
{
  "message": "The given data was invalid.",
  "errors": {
    "estrellas": [
      "The estrellas field is required."
    ]
  }
}
```
</ResponseExample>
